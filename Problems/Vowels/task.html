Vowels

There is a program that reads a letter from the input. It must print the number of this letter in the alphabet only if it's a vowel. If the input letter is not a vowel, the program must print 0.

Do not consider the letter 'y' as a vowel.


Sample Input:
z

Sample Output:
0



Memory limit: 256 MB
Time limit: 8 seconds
